# Access-Style Stock Control Demo

This sample is a static front-end demo for a small single-user stock control
program. It is designed to match client requests for a simple Microsoft
Access-style database workflow before building the real app.

Live demo:

https://zhuzhu21210-commits.github.io/upwork-automation-portfolio/stock-control-access-demo.html

## What It Demonstrates

- stock item master fields
- stock increase and decrease entry form
- reason fields for stock movement
- current stock and stock value reporting
- negative-stock visibility
- searchable report table by category, item, supplier, or location

## Best-Fit Client Projects

- Microsoft Access-style stock control program
- small warehouse or storeroom inventory tracker
- single-user inventory database
- first milestone before a larger inventory app

## Notes

This is a visual and interaction demo, not a production database. For a paid
project, the first milestone can turn this structure into Microsoft Access,
SQLite, Excel/CSV, or another local database format based on the client's
preferred workflow.
