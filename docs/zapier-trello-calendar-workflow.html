Questions I Confirm First

Trigger Rules

  • Run when a card is created, moved to a list, or assigned a label?
  • Update existing events when Trello cards change?
  • Which timezone should be used for calendar events?

Error Handling

  • Skip cards without due dates or log them for review?
  • Email a summary of failed cards or store it in Google Sheets?
  • Handle Xero and Drive files now or in a second workflow?

Delivery

The first milestone should end with one working automation, test examples, clear assumptions, and concise handoff notes. This keeps the project useful quickly and avoids turning a small Zapier build into an unbounded operations rebuild.
