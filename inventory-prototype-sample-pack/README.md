# Inventory Prototype Sample Pack

Use this pack when a client asks what input is needed for a first inventory,
stock-list, or Excel/CSV automation milestone.

Live landing page:

https://zhuzhu21210-commits.github.io/upwork-automation-portfolio/inventory-prototype.html

## Files

- `inventory_template.csv`
  - example product/SKU stock list
- `stock_movements_template.csv`
  - example stock-in and stock-out rows
- `desired_report.md`
  - plain-English report specification

## Client Instructions

For a real project, the client can send:

1. one sanitized inventory or product export
2. one example of stock changes, if used
3. the fields they want in the final report
4. the preferred output format: CSV, Excel, Python script, Access-style database, or web page

The first milestone should stay focused: one input format, one workflow, and one
reviewable output.
